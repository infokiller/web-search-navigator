Keybindings

Help
All keybindings should be specified in Mousetrap format. Examples:
  • a
  • z y
  • ctrl+a
  • command+a
  • a, ctrl+b, z y, command+c - multiple shortcuts that will be treated equivalently
Special keys names: backspace, tab, clear, enter, return, esc, escape, space, up, down, left, right, home, end, pageup, pagedown, del, delete, and f1 through f19. In order to disable a keybinding, delete its keybinding in the textbox. Note that not all search engines support all the keybindings.
